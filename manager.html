How to use this extension:

On this page you can create and remove tab groups, see the tabs within each group, and drag tabs from one group to another group. Click on the 'Open Tab Group' button to open all the tabs in that group and close the tabs in the previously open group.

This extension is maintained by pjh5000. The underlying source code can be found at: https://github.com/pjh5000/TabGroupsForChrome

The code is based on the original TabsFolder extension by Conchilicultor, available from: https://github.com/Conchylicultor/TabsFolder.

Tab icons made by Picol from www.flaticon.com, licensed by CC 3.0 BY.

I hope you find this extension useful.
